# 项目概述

## 项目定位
ComfyUI 的增强代理层，提供统一管理、AI 辅助编辑和版本管理能力。

## 核心价值
- 统一管理多个 ComfyUI 服务实例
- AI Agent 自然语言辅助编辑工作流
- 工作流版本管理和会话历史记录

## 系统架构
```
用户界面 (Web + AI Chat)
        ↓
代理系统 (认证/存储/AI/管理)
        ↓
ComfyUI 服务集群
```

## 用户角色
- **管理员**：管理服务实例、用户权限、系统监控
- **普通用户**：创建工作流、使用 AI 辅助、管理会话

## 技术栈
- **后端**：Spring Boot 3.x + PostgreSQL + Redis
- **前端**：Vue 3 + Element Plus
- **AI**：LLM API 集成
- **通信**：WebSocket + RESTful API

## 项目范围
**包含**：用户认证、服务管理、工作流 CRUD、AI 对话编辑、版本管理
**不包含**：ComfyUI 核心修改、自定义节点开发、实时协作
