# ComfyUI Pilot - 项目概述

## 项目定位
ComfyUI 的增强代理层，提供统一管理、AI 辅助编辑和版本管理能力。

## 核心价值
- 统一管理多个 ComfyUI 服务实例
- AI Agent 自然语言辅助编辑工作流
- 工作流版本管理和会话历史

## 系统架构
```
用户界面 (Web + AI Chat)
        ↓
代理系统 (认证/存储/AI/管理)
        ↓
ComfyUI 服务集群
```

## 用户角色
- **管理员**: 管理服务、用户、Agent、AI模型
- **普通用户**: 创建工作流、使用 AI 辅助、管理会话

## 技术栈
- **后端**: Spring Boot 3.x + PostgreSQL + Redis + MyBatis-Plus
- **前端**: Vue 3 + Element Plus
- **AI**: 多模型提供商集成 (OpenAI/Anthropic/国产模型)
- **通信**: WebSocket + RESTful API

## 核心模块
1. **用户模块**: 用户认证、权限管理
2. **服务模块**: ComfyUI 服务管理
3. **工作流模块**: 工作流 CRUD、版本管理
4. **会话模块**: AI 对话、消息管理
5. **Agent模块**: Agent 配置、调度、执行
6. **AI模型模块**: 模型提供商、模型配置、调用管理
7. **资源模块**: 文件上传、存储管理

## 项目范围
**包含**: 用户认证、服务管理、工作流 CRUD、AI 对话编辑、版本管理、Agent 管理、模型管理
**不包含**: ComfyUI 核心修改、自定义节点开发、实时协作
